## House style

- Hand-authored. Vanilla HTML / CSS / JS where it fits; TypeScript or Python where it earns it.
- No build steps unless the project genuinely needs one.
- Consent-first on anything that touches a user.
- Readable commits, small PRs, plain-English READMEs.
